# Table Filtering
Use _filter-options_ prop to filter items in a table.

## Filter comparisons:

| Value   | Description              |
|---------|--------------------------|
| between | Between                  |
| in      | In                       |
| =       | Equals                   |
| !=      | Does not equal           |
| >       | Greater than             |
| >=      | Greater than or equal to |
| <       | Less than                |
| <=      | Less than or equal to    |
